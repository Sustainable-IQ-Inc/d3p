# API Reference

## Overview

Technical documentation for developers working with the D3P platform.

## Endpoints

### Project APIs
- **GET** `/api/projects` - List all projects
- **POST** `/api/projects` - Create a new project
- **PUT** `/api/projects/:id` - Update project details

### Upload APIs
- **POST** `/api/upload` - Upload energy model files
- **GET** `/api/upload/:id` - Get upload status

### Data Analytics APIs
- **GET** `/api/data/eeu` - Retrieve energy usage data
- **GET** `/api/data/summary` - Get project summaries
